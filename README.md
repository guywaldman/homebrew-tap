# Guy Waldman's Homebrew Tap

This is a repository to store all my [Homebrew](http://brew.sh) taps and formulae.

## Update Glue formula

Run:

`uv run scripts/update_glue.py`

Or via just:

`just update-glue`

Or for a specific tag:

`uv run scripts/update_glue.py --tag v0.1.1`

Or via just:

`just update-glue tag=v0.1.1`
